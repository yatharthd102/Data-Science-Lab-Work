# Project Title
Student Performance Dashboard
## Description
An interactive data visualization dashboard that analyzes student scores
across 5 subjects: Math, Science, English, History, and Computer.
The dashboard provides insights into grade distribution, subject averages,
top performers, and score correlations.
## Dataset
- Source: Synthetically generated (random data)
- 50 students with scores in 5 subjects (range: 40-100)
- Features: Student ID, subject scores, total, average, grade
## Steps Performed
1. Data Generation - Created realistic student score dataset using NumPy
2. Feature Engineering - Calculated Total, Average, Grade columns
3. Exploratory Data Analysis - Computed per-subject statistics
4. Visualization - Built 6-panel dashboard with Matplotlib
## Results
- Average score across all subjects: ~70
- Most students fall in C and B grade range
- Math and Computer show highest average scores
- Clear positive correlation between Math and Science scores
## Tools Used
- Python, NumPy, Pandas, Matplotlib
## Conclusion
The dashboard clearly shows normal score distribution with most students
performing in the C-B range. Subject-wise analysis helps identify where
students need more attention.
